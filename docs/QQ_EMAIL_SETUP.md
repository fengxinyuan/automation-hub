# QQ 邮箱授权码获取指南

## 什么是授权码？

QQ 邮箱的授权码是用于第三方客户端登录的专用密码，**不是你的 QQ 密码**。

## 获取步骤

### 1. 登录 QQ 邮箱
访问：https://mail.qq.com/

### 2. 进入设置
点击顶部的 **设置** → **账户**

### 3. 开启 SMTP 服务
找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务** 区域

### 4. 开启服务
- 找到 **SMTP服务** 或 **IMAP/SMTP服务**
- 点击 **开启** 按钮
- 按照提示发送短信验证（需要绑定手机）

### 5. 获取授权码
- 验证成功后，系统会显示一个 **授权码**
- 这是一个 16 位的字符串，类似：`abcdefghijklmnop`
- **重要**：请妥善保存这个授权码，关闭后无法再次查看

## 注意事项

1. ⚠️ 授权码不是 QQ 密码，是专门用于第三方登录的
2. ⚠️ 授权码只显示一次，关闭后需要重新生成
3. ⚠️ 如果忘记授权码，可以重新生成新的授权码
4. ⚠️ 生成新授权码后，旧的授权码会失效

## QQ 邮箱 SMTP 配置信息

```
SMTP 服务器：smtp.qq.com
SMTP 端口：587（TLS）或 465（SSL）
安全连接：TLS 或 SSL
```

## 常见问题

### Q: 没有收到验证短信？
A: 检查手机是否绑定正确，可以尝试重新发送

### Q: 提示"请先绑定密保手机"？
A: 需要先在 QQ 安全中心绑定手机号码

### Q: 授权码不显示？
A: 刷新页面重试，或者清除浏览器缓存后重新操作

### Q: 使用授权码后还是登录失败？
A: 检查是否复制了完整的授权码（16位），注意不要有空格

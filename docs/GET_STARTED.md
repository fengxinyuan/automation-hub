## 🚀 两种使用方式

### 方式1：简化模式（立即可用）

**无需任何配置**，直接运行：

```bash
cd /root/auto-checkin
python3 src/main.py --site linuxdo
```

**你会得到**：
- ✅ 30个帖子信息（最新20 + 热门10）
- ✅ 按热度智能排序
- ✅ 简单内容提取
- ⚠️ 没有 AI 深度分析

**运行时间**: 约2-3分钟

---

### 方式2：AI 完整模式（推荐）⭐

启用 AI 获得最佳体验！

#### 步骤1：获取 OpenAI API Key

访问 https://platform.openai.com/api-keys
- 注册/登录 OpenAI 账号
- 创建新的 API Key
- 复制保存

#### 步骤2：配置

```bash
cd /root/auto-checkin

# 创建配置文件
cp .env.example .env

# 编辑配置
nano .env
```

添加你的 API Key：

```bash
# OpenAI API Key
OPENAI_API_KEY=sk-your-actual-key-here

# 使用的模型（可选，默认 gpt-3.5-turbo）
AI_MODEL=gpt-3.5-turbo
```

保存并退出（Ctrl+X, Y, Enter）

#### 步骤3：运行

```bash
python3 src/main.py --site linuxdo
```

**你会得到** ⭐：
- ✅ 30个帖子全面分析
- ✅ AI 推荐5个最相关话题（带理由和相关度）
- ✅ AI 深度分析3个热门帖子（摘要+要点+标签）
- ✅ 高质量内容总结
- ✅ 情感倾向分析

**运行时间**: 约3-5分钟
**费用**: 约 $0.01-$0.02/次
